# GUA Player 更新日誌  
## v1.0.3 版本亮點與修復

### 🌟 新功能與優化 (Features & Enhancements)
- **多國語言介面 (Interface Language Switcher)**
  - 主程式介面與設定視窗現在支援即時語言切換。
  - 支援繁體中文 (Traditional Chinese) 與 英文 (English)。
  - 在「設定 (Settings)」視窗的右上方可輕易切換，程式會即刻套用全新語言，無須重新啟動。

## v1.0.2 & v1.0.1 累積更新重點
- **全新 Apple Music 支援**：由高度相容且穩定無 DRM 干擾的 Apple Music 替換先前的 Spotify，提供無縫的聽歌與控制體驗。
- **全平台暴力音量引擎**：徹底重寫底層切換邏輯，現在不管音樂由哪個背景標籤發出，GUA Player 都會強制鎖住追蹤您的音量設定，確保聽覺體驗不再忽大忽小。
- **新增音量全域快捷鍵**：支援 `Ctrl` + `Shift` + `Up/Down` 上下方向鍵調節音量。
- **解決跨平台打包圖示崩潰問題**：徹底修復了 GitHub Actions 編譯 macOS / Linux 發布檔時的編碼阻礙。
- **多國語言安裝介面**：開啟 Windows `.exe` 安裝檔時第一時間跳出繁/英語系供您選擇。
